🚀 Soroban Smart Contract Deployer

📋 Instructions:

  1. Copy the invoice contract code from contracts/invoice_contract_ready.rs
  2. Paste it in the Soroban Playground: https://soroban.stellar.org
  3. Click "Build" to compile to WASM
  4. Download the WASM file
  5. Repeat for registry contract
  6. Use the commands below to deploy

🔧 Deployment Commands

1. Deploy Invoice Contract

stellar contract deploy --wasm invoice_contract.wasm --source YOUR_SECRET_KEY --network testnet

2. Deploy Registry Contract

stellar contract deploy --wasm registry_contract.wasm --source YOUR_SECRET_KEY --network testnet

3. Initialize Invoice Contract

stellar contract invoke --id INVOICE_CONTRACT_ADDRESS --source YOUR_SECRET_KEY --network testnet -- initialize --usdc_address GBBD47IF6LWK7P7MJVSCUFI274NM5YCFA247JEC2HGJLMD2BDQD2XUJR --registry_address REGISTRY_CONTRACT_ADDRESS

4. Initialize Registry Contract

stellar contract invoke --id REGISTRY_CONTRACT_ADDRESS --source YOUR_SECRET_KEY --network testnet -- initialize --admin YOUR_PUBLIC_KEY
